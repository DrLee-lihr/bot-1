
# `newbie`

## `newbie`
:octicons-unlock-24: 公开

用法：`~newbie`

此指令可以让你查询设置的起始Wiki的新用户日志。

!!! example "示例"
    本指令返回的结果示例：
    ```
    这是当前的新人列表：
    User:Foxiagouhe
    User:梦流年
    User:Wing0805
    User:Julie130
    User:陳榮森
    ...仅显示前5条内容
    ```

!!! warning "警告"
    某些网站已经被列入黑名单，使用此命令无法获取某些网站的信息。即使网站不在黑名单内，返回的内容仍将经过小可云智能过滤，请不要作死。
