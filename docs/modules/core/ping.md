# `ping`

:octicons-file-code-24: 开源

此模块可以让你得知机器人是否在线及获取系统信息。

## `ping`
:octicons-unlock-24: 公开

使用此指令会得到机器人的运行信息。

用法：`~ping`

!!! example "示例 ping 信息"
    这是一份示例的 ping 信息。
    ```
    Pong!
    系统运行时间：2021-01-26 22:26:46
    当前CPU使用率：1.1%
    物理内存：3807M 使用率：42.6%
    Swap内存：2042M 使用率：12.7%
    ```
